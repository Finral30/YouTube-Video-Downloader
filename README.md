# YouTube Video Downloader

A modern YouTube Video & Audio Downloader built using Python, CustomTkinter, FFmpeg, and pytubefix.

This application allows you to download high-quality YouTube videos and audio with a clean desktop GUI experience on Windows 10 and Windows 11.

---

## Features

- Download YouTube videos in multiple qualities
- Download audio as MP3
- Automatic video + audio merging using FFmpeg
- Modern CustomTkinter GUI
- Download progress bar
- Download cancellation support
- Download history
- Automatic fallback stream selection
- Threaded downloading (UI stays responsive)
- Portable EXE support
- Windows 10 & 11 compatible
